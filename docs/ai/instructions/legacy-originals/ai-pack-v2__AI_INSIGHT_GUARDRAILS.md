# AI Insight Guardrails

## Rule 1
Never present speculation as race fact.

## Rule 2
Every insight card must carry:
- timestamp
- target entity
- source context
- rationale
- confidence/certainty band

## Rule 3
Mark whether the insight is:
- rule-based inference
- model-based prediction
- uncertain hypothesis

## Rule 4
If source fidelity is degraded, the insight must reflect reduced confidence.

## Rule 5
Prefer explainable heuristics over opaque model claims in early product stages.

## Rule 6
Do not fabricate causal explanations when only correlation is present.
